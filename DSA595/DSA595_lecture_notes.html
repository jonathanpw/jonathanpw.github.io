1. Random Variables

A random variable is a measurable function X : Ω → ℝ.

Discrete random variable
Takes countably many values. Described by a probability mass function (pmf).
Examples: Bernoulli(p), Binomial(n, p), Poisson(λ)
Continuous random variable
Takes all values on one or more intervals. Described by a probability density function (pdf).
Examples: Uniform(a, b), Normal(μ, σ²)

Joint Distributions

The joint distribution of random variables X and Y is described by the joint pmf/pdf fX,Y(x, y).

Bayes' Rule

The conditional distribution of X given Y = y is:

fX|Y(x | y) := fX,Y(x, y) fY(y) so long as fY(y) > 0.

2. Bayesian Inference

Suppose data x₁, …, xₙ are observations of i.i.d. random variables X₁, …, Xₙ from a probability distribution whose pmf/pdf belongs to the parametric family { fX|Θ(x | θ) : θ some parameter(s) }.

If we have prior information about the true θ* in the form of a prior pmf/pdf ΠΘ(θ), we can define the posterior distribution:

ΠΘ|X₁,…,Xₙ(θ | x₁,…,xₙ) := fX₁,…,Xₙ|Θ(x₁,…,xₙ | θ) · ΠΘ(θ) ∫ fX₁,…,Xₙ|Θ(x₁,…,xₙ | ϑ) · ΠΘ(ϑ) dϑ

Short-hand notation:

Π(θ | x₁,…,xₙ) = f(x₁,…,xₙ | θ) · Π(θ) ∫ f(x₁,…,xₙ | ϑ) · Π(ϑ) dϑ

3. Conjugate Priors — Bernoulli Example

Setup: X₁,…,Xₙ i.i.d. ~ Bernoulli(p), prior on p is Beta(a, b). Find the posterior of p.

Likelihood

f(x₁,…,xₙ | p) = ∏ᵢ₌₁ⁿ p^xᵢ · (1−p)^(1−xᵢ) = p^(Σxᵢ) · (1−p)^(n − Σxᵢ)

Prior

Π(p) = Γ(a+b) / [Γ(a)·Γ(b)] · p^(a−1) · (1−p)^(b−1) for a, b > 0

Posterior

Multiplying likelihood × prior and simplifying (the normalizing constant cancels):

Π(p | x₁,…,xₙ) = Beta(a + Σxᵢ, b + n − Σxᵢ)
Conjugate prior: Posterior distributions in the same parametric family as the prior are called conjugate. Here Beta is conjugate to the Bernoulli/Binomial likelihood.

4. Non-Conjugate Prior — Normal Example

Setup: X₁,…,Xₙ i.i.d. ~ N(μ, 1), prior μ ~ Uniform(a, b).

Working through the algebra, the posterior is a truncated normal:

Π(μ | x₁,…,xₙ) = (2π · 1/n)−½ · exp[−½n(μ − x̄)²] · 𝟙{μ ∈ (a,b)} Φ((b − x̄) / (1/√n)) − Φ((a − x̄) / (1/√n))

where Φ(·) is the CDF of the standard Normal distribution.

Flat prior: Setting (a, b) = ℝ gives the improper "flat" prior Π(μ) ∝ 1. The resulting posterior is proper and equals N(x̄, 1/n), analogous to the sampling distribution of x̄, i.e., N(μ, 1/n).

5. Random Number & Variable Generation

We assume software can generate random instances from Uniform(0,1). The question is: how do we sample from an arbitrary continuous distribution?

Probability Integral Transform

Let X be a continuous random variable with CDF F_X. Then:

Y := F_X(X) ~ Uniform(0,1)

Proof sketch: For y ∈ (0,1),

F_Y(y) = P(Y ≤ y) = P(F_X(X) ≤ y) = P(X ≤ F_X⁻¹(y)) = F_X(F_X⁻¹(y)) = y

This is the CDF of Uniform(0,1). Therefore, if we generate U ~ Uniform(0,1) and F_X is continuous, then X = F_X⁻¹(U) has the distribution of X.

6. Metropolis–Hastings (MH) Algorithm

Suppose Π(θ|x) is a posterior density of interest but we do not know its normalizing constant. We know only h(θ|x) where Π(θ|x) = c · h(θ|x) ∝ h(θ|x).

Assume q(θ) is a proposal density with supp{h(θ|x)} ⊆ supp{q(θ)}.

MH Algorithm — Independent Sampling

Algorithm: MH (Independent Sampling)

  1. Start with current state θ(t).
  2. Randomly sample candidate θ* ~ q.
  3. Compute acceptance probability:
    ρ(θ*, θ(t)) = min { h(θ*|x) / q(θ*) h(θ(t)|x) / q(θ(t)) , 1 }
  4. Set θ(t+1) = θ* with probability ρ; else θ(t+1) = θ(t).

MH Algorithm — Gaussian Random Walk

Use the proposal distribution q(θ*|θ(t)) ~ N(θ(t), τ²) where τ > 0 controls the scale of proposals.

Algorithm: MH (Gaussian Random Walk)

  1. Start with current state θ(t).
  2. Sample θ* = θ(t) + Z, where Z ~ N(0, τ²).
  3. Compute acceptance probability:
    ρ(θ*, θ(t)) = min { h(θ*|x) h(θ(t)|x) , 1 } (q cancels by symmetry of N(0,τ²))
  4. Set θ(t+1) = θ* with probability ρ; else θ(t+1) = θ(t).

Values of θ are accepted in direct proportion to the density Π(θ|x). The normalizing constant c cancels in the ratio, so it need never be computed.

7. Markov Chain Monte Carlo (MCMC)

The MH algorithm is the canonical example of a broader class of algorithms called Markov chain Monte Carlo (MCMC) methods. MCMC is primarily what constitutes "Bayesian computation."

A central challenge in MCMC is balancing acceptance rate vs. mixing:

  • Too few acceptances → algorithm converges slowly.
  • Too many acceptances → accepted proposals are highly dependent.
  • Target acceptance rate is roughly 20–50% for adequate mixing.

One approach to controlling acceptance rate is to use a random walk proposal and tune the scale parameter τ².

8. Assessing MCMC Convergence — Gelman–Rubin Statistic

Convergence is most often assessed via visual inspection of the trace plot, but more rigorous diagnostic metrics have been proposed.

Gelman–Rubin (GR) Statistic

Run J chains of an MCMC algorithm, each with a different initial value. Denote chain j ∈ {1,…,J} by x₁(j),…, x_L(j) (post burn-in).

Sample mean of chain j
x̄ⱼ := (1/L) Σᵢ₌₁ᴸ xᵢ(j)
Grand mean (mean of chain means)
x̄* := (1/J) Σⱼ₌₁ᴶ x̄ⱼ
Between-chain variance B (scaled by L)
B := L/(J−1) · Σⱼ₌₁ᴶ (x̄ⱼ − x̄*)²
Within-chain variance W
W := (1/J) Σⱼ₌₁ᴶ { (1/(L−1)) Σᵢ₌₁ᴸ (xᵢ(j) − x̄ⱼ)² }
GR statistic R̂
R̂ := ((L−1)/L) · W + (1/L) · B W
Interpretation: As L → ∞, B/L → 0 and R̂ → 1 if the chain has converged. It has been argued in the literature that R̂ ∈ [1, 1.1] indicates convergence.

9. Gibbs Sampling

Suppose Π(θ₁,…,θₚ | y) is a posterior with unknown normalizing constant, but we can derive the fully specified conditional densities:

Π(θ₁ | θ₂,…,θₚ, y) ⋮ Π(θₚ | θ₁,…,θₚ₋₁, y)

Algorithm: Gibbs Sampler

  1. Start with current state θ(t) = (θ₁(t),…, θₚ(t)).
  2. Sample each component from its full conditional:
    θ₁(t+1) ~ Π(· | θ₂(t),…,θₚ(t), y) ⋮ θₚ(t+1) ~ Π(· | θ₁(t),…,θₚ₋₁(t), y)

Why does this work? Construct the MH ratio for θ₁ | θ₂,…,θₚ, y using Π(θ₁ | θ₂,…,θₚ, y) as the proposal. The ratio evaluates to 1, so every proposal is accepted — Gibbs sampling is a special case of MH with acceptance probability always equal to 1.

10. Monte Carlo Integration

Let X ~ f_X. For any f_X-integrable function h : X → ℝ:

∫ h(x) f_X(x) dx = E[h(X)] ≈ (1/n) Σᵢ₌₁ⁿ h(xᵢ)

for an i.i.d. observed sample x₁,…,xₙ. Moreover, for a bounded integral on [a,b]:

∫ₐᵇ h(x) f_X(x) dx ≈ (1/n) Σᵢ₌₁ⁿ h(xᵢ) · 𝟙{a ≤ xᵢ ≤ b}

Importance Sampling

When it is analytically intractable to integrate some function g, write:

∫ g(x) dx = ∫ [g(x)/f_X(x)] · f_X(x) dx = E[g(X)/f_X(X)] ≈ (1/n) Σᵢ₌₁ⁿ g(xᵢ)/f_X(xᵢ)

for an i.i.d. sample x₁,…,xₙ, so long as supp(g) ⊆ supp(f_X).
